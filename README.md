# AI Career Recommendation System

##  Overview

This project is an AI-based system that recommends careers based on user
skills, interests, and personality traits. It uses machine learning
techniques to provide personalized suggestions.

##  Features

-   Personalized career recommendations
-   Machine learning-based prediction
-   Easy-to-use interface
-   Scalable design

##  Technologies Used

-   Python
-   pandas
-   scikit-learn

##  How It Works

1.  Dataset is created and loaded
2.  Data is encoded using OneHotEncoder
3.  Model is trained using Decision Tree
4.  User inputs are taken
5.  Model predicts suitable career

## ▶️ Installation & Setup

``` bash
pip install pandas scikit-learn
python career_ai.py
```

##  Project Structure
